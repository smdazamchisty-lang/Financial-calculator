# 🛡️ WAR_CHEST.AI

A brutalist financial calculator for high-volatility environments. 
Built with **Vibe Coding** principles: Speed, intuition, and raw utility.

## The Vibe
- **No Charts:** Charts are for peacetime. We use numbers.
- **No Ads:** Survival is ad-free.
- **Total Privacy:** Logic runs client-side. Your war chest is your business.

## Installation
`git clone survival-calculator`
`npm install && npm run dev`

## Logic Flow
Input ($X) -> Adaptive Wartime Multiplier -> Tiered Allocation -> Survival.
